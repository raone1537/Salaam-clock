Salaam Clock - Pi Browser Prayer Times App
-----------------------------------------

Files:
- index.html
- style.css
- app.js

Defaults:
- Cities: Makkah, Madina (Saudi Arabia)
- Data source: Aladhan API (https://api.aladhan.com)
- Pi SDK script included: https://sdk.minepi.com/pi-sdk.js

Notes for deployment:
1. Upload the folder to a static host (Vercel, Netlify, Firebase or your server).
2. Set the app URL in Pi Developer Portal and configure payment/merchant settings to enable Pi payments.
3. Test inside Pi Browser (app may need Pi SDK enabled).
4. Adjust donation amount and payment payload as needed.
